Kostka number

Kostka number, \(K_{\lambda\mu}\), is equal to the number of Young tableaux of shape \(\lambda\) and weight \(\mu\). Let \(S\) be a set. A matrix \(m : S\times S\to \{1,2, \dots ,\infty\}\) is called a Coxeter matrix if it satisfies $$m(x,y) = m(y,x)$$ $$m(x,y) = 1\text{ implies }x=y$$

Some Vscode Command

How to switch from $ $ to \( \)

Type out the following code in the find field: 

\$([^\$]+)\$

Then type out the following code in the replace field: 

\\($1\\)

How to switch from \text to \textbf

Type out the following code in the find field: 

\\ text

Then type out the following code in the replace field: 

\\textbf

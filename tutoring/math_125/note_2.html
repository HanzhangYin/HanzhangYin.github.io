Limits at Infinity

Consider limits where \(x\) approaches \(+\infty\) or \(-\infty\).

Constant Cases

Theorem. For all constant numbers \( c \), \[ \lim_{x \to \infty} c = c \qquad \text{ and }\qquad \lim_{x \to -\infty} c = c. \] Example. \[ \lim_{x \to \infty} 23 = 23\qquad \text{ and }\qquad\lim_{x \to -\infty} 100 = 100. \]

\(x\) Approaches \(+\infty\)

Theorem. For all \( n > 0 \), \[ \lim_{x \to \infty} x^n = \infty \quad \text{and} \quad \lim_{x \to \infty} x^{-n} = \lim_{x \to \infty} \frac{1}{x^n} = 0 \] Example. \[ \lim_{x \to \infty} x^3 = \infty \quad \text{and} \quad \lim_{x \to \infty} x^{-3} = \lim_{x\to \infty}\frac{1}{x^3} = 0 \]

\(x\) Approaches \(-\infty\)

Theorem. If \( n \) is a positive whole number, \[ \lim_{x \to -\infty} x^n = \begin{cases} \infty & \text{if } n \text{ is even} \\ -\infty & \text{if } n \text{ is odd} \end{cases} \quad \text{and} \quad \lim_{x \to -\infty} x^{-n} = \lim_{x \to -\infty} \frac{1}{x^n} = 0 \]

Example. \[ \lim_{x \to -\infty} x^3 = -\infty \quad \text{and} \quad \lim_{x \to -\infty} x^4 = \infty \] \[ \lim_{x \to -\infty} x^{-3} = \lim_{x\to -\infty}\frac{1}{x^3} = 0, \quad \text{and} \quad \lim_{x \to -\infty} x^{-4} = \lim_{x\to -\infty}\frac{1}{x^4} = 0 \]

Limits at Infinity of a Rational Function

Here the rational functions are of the form polynomial over polynomial.

Example \[ \frac{3x^2 + 2x + 1}{x^2 + 1}, \quad \frac{x^5 + 2x + 1}{x^7 + x^4 + x^2 + 3}, \quad \frac {2x^2 + 1}{x + 1} \] Theorem.The asymptotic behavior of a rational function depends only on the leading terms of its numerator and denominator. If \( a_n, b_m \neq 0 \), then \[ \lim_{x \to \infty} \frac{a_n x^n + a_{n-1} x^{n-1} + \cdots + a_0}{b_m x^m + b_{m-1} x^{m-1} + \cdots + b_0} = \frac{a_n}{b_m}\cdot(\lim_{x \to \infty} x^{n-m}). \] \[ \lim_{x \to -\infty} \frac{a_n x^n + a_{n-1} x^{n-1} + \cdots + a_0}{b_m x^m + b_{m-1} x^{m-1} + \cdots + b_0} = \frac{a_n}{b_m}\cdot(\lim_{x \to -\infty} x^{n-m}). \]


Intermediate Value Theorem

A Fundamental Question: How do we know that a continues function \(f(x) = 0\) has a root?

One way is to use the Intermediate Value Theorem.

The Intermediate Value Theorem (IVT) says that a continuous function cannot skip values.

The Intermediate Value Theorem states that

  • If \( f \) is continuous on \([a, b]\)
  • \( L \) is any number between \( f(a) \) and \( f(b) \)

Then there exists a number \( c \) in \((a, b)\) such that \( f(c) = L \).

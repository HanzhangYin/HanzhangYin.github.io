7. Expand the Product

\[ (2x + y)(3x + 5) \]

\(\textbf{Solution. }\) We can use distribution to expand the product. \[ \begin{align} (2x + y)(3x + 5) &= 2x(3x + 5) + y(3x + 5) \\ &= 2x(3x) + 2x(5) + y(3x) + y(5) \\ &= 6x^2 + 10x + 3xy + 5y \end{align} \] Box method is another way to expand the product.


8. Factoring

Version 1

\[ 3x^2 - 15x + 18 \]


\(\textbf{Solution. }\) We can first identify the greatest common factor (GCF) of the polynomial, which is 3. \[ \begin{align} 3x^2 - 15x + 18 &= 3(x^2 - 5x + 6)\\ \end{align} \] Then we are going to find \(r, s\) such that \[ \begin{align} r + s &= -5 \\ rs &= 6 \end{align} \] Hence, we have \(r = -3, s = -2\). Therefore, we can factor the polynomial as \[ \begin{align} 3x^2 - 15x + 18 &= 3(x - 3)(x - 2) \end{align} \]


Version 2

\[ 2x^3 + 7x^2 + 3x \]

\(\textbf{solution}\). Similarly, we can first identify the GCF of the polynomial, which is \(x\). \[ \begin{align} 2x^3 + 7x^2 + 3x &= x(2x^2 + 7x + 3) \end{align} \] For the quadratic polynomial \(2x^2 + 7x + 3\), we are going to use \(AC\) method to factor it. We can first find \(r, s\) such that \[ \begin{align} r + s &= 7 \\ rs &= 6 \end{align} \] Hence, we have \(r = 6, s = 1\). Therefore, we can replace \(7x\) with \(6x + x\). \[ \begin{align} 2x^2 + 7x + 3 &= 2x^2 + 6x + x + 3 \\ &= 2x(x + 3) + 1(x + 3) \\ &= (2x + 1)(x + 3) \end{align} \] Therefore, we can factor the polynomial as \[ \begin{align} 2x^3 + 7x^2 + 3x &= x(2x + 1)(x + 3) \end{align} \]


9. Multiply Rational Expression

\[ \frac{x^2 - 2x + 1}{x^2 +2x + 1} \cdot \frac{5x + 5}{2x - 2} \]

\(\textbf{solution. }\) We can first factor the numerator and denominator of the rational expression. \[ \begin{align} \frac{x^2 - 2x + 1}{x^2 +2x + 1} \cdot \frac{5x + 5}{2x - 2} &= \frac{(x - 1)^2}{(x + 1)^2} \cdot \frac{5(x + 1)}{2(x - 1)} \\ &= \frac{(x - 1)( x - 1 )}{(x + 1)(x + 1)}\frac{5(x + 1)}{2(x - 1)} \\ &= \frac{(x - 1)^2}{(x + 1)^2}\frac{5(x + 1)}{2(x - 1)} \end{align} \]

Now we can combine the numerator and denominator of the rational expression. \[ \begin{align} \frac{(x - 1)^2}{(x + 1)^2}\frac{5(x - 1)}{2(x + 1)} &= \frac{5(x - 1)^2(x+1)}{2(x + 1)^3} \end{align} \]

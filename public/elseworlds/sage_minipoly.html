Some commands on homebrew

To check packages which could be updated through brew, we can use the following codes:

brew update

This command fetches the latest version of Homebrew and its formulae (the package descriptions) from GitHub.

To upgrade all packages, use the following code

brew upgrade

To see all the packages installed on brew: use the following code:

brew list
If you want to see the versions of packages, use the following code: 
brew list --version

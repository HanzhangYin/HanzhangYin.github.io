Ehrhart Theory & Convex Polytope

Lattice Polygon

\(\textbf{Definition. }\)A lattice polygon in \(\mathbb{R}^2\) is a polygon where vertices have integer coordinates.

Is there something like Pick's Theorem for n-dimensional lattice polyhedra?

The answer is no. There is no formula for the volume of a lattice polyhedron in \(\mathbb{R}^n\).

Reeve Tetrahedron

Reeve Tetrahedron is a tetrahedron with vertices \((0,0,0), (1,0,0), (0,1,0), (1,1,r)\) for some integer \(r\in\mathbb{Z}\).

There are only \(4\) lattice points.

\(\textbf{Theorem. }\)The volume of Reeve Tetrahedron is \(\frac{r}{6}\).


Convex Hull

\(\textbf{Definition. }\)The convex hull of a set \(S\) is the smallest convex set containing \(S\).

\(\textbf{Example. }\)The convex hull of \(\{(0,0), (1,0), (0,1)\}\) is the triangle with vertices \((0,0), (1,0), (0,1)\).


Convex Polytope

\(\textbf{Definition. }\)A convex polytope is a convex hull of finitely many points in \(\mathbb{R}^n\). \[ P = \text{conv}\{v_1, v_2, \cdots, v_k\} = \{\sum_{i=1}^m\lambda_iv_i:\;\lambda_i\geq 0 and \sum\lambda_i = 1\} \]

\(\textbf{Alternate Definition. }\)A polytope is a bounded intersection of finitely many half-spaces and hyperplanes in \(\mathbb{R}^n\).

\(\textbf{Example. }\)A unit cube is a convex polytope. \[ [0, 1]^n = \text{conv}\{(x_1, x_2, \cdots, x_n)\in\mathbb{R}^n:\;x_i\in\{1, 0\}\}=\{(x_1, x_2, \dots, x_n):\;0\leq x_i\leq 1\}. \]


Dimension

\(\textbf{Definition. }\)The dimension of a polytope \(P\) is the dimension of the affine subspace \[ \text{span}(P) = \{v_i+\lambda(v_i-v_j):\;v_i, v_j\in P, \;\lambda\in\mathbb{R}\}. \]


Lattice Polytope

\(\textbf{Definition. }\)P is a lattice polytope if it can be expressed as the convex hull of integer coordinate points.

Dilate

\(\textbf{Definition. }\)The dilate of a polytope \(P\) by a factor \(k\) is the polytope \(kP\) obtained by multiplying all coordinates of \(P\) by \(k\).

\(\textbf{Example. }\)The dilate of a unit cube by a factor \(k\) is a cube with side length \(k\).

Lattice Point Enumerator

\(\textbf{Definition. }\)The lattice point enumerator of a polytope \(P\) is the function \(L_P(k)\) that counts the number of lattice points in the dilate \(kP\). \[ L_P(k) = \text{#}(kP\cap \mathbb{Z}^n), or \] \[ L_P(k) = \text{#}(P\cap \frac{1}{k}\mathbb{Z}^n). \]

\(\textbf{Example. }\)The lattice point enumerator of a unit cube \([0, 1]^n\) is \(L_P(k) = (k+1)^n\) when \(k\in\mathbb{Z}^+\).

Standard Simplex Example

\(\textbf{Definition. }\)The standard simplex \(\Delta_n\) is the convex hull of the standard basis vectors in \(\mathbb{R}^n\). \[ \Delta_n = \text{conv}\{e_1, e_2, \cdots, e_n\} = \{(x_1, x_2, \cdots, x_n):\;x_i\geq 0, \sum_{i=1}^nx_i \leq 1\}. \]

\(\textbf{Example. }\)Then for \(t\in\mathbb{Z}^+\), the lattice point enumerator of \(\Delta_n\) is \[ t\cdot \Delta_n = \{(x_1, x_2, \cdots, x_n):\;x_i\geq 0, \sum_{i=1}^nx_i \leq t\}. \]

\[ t\cdot \Delta_n\cap \mathbb{Z}^n = \{\text{non negative integer solutions to }x_1+x_2+\cdots+x_n \leq t\}. \] To count this, we need to add a slack variable \(x_{n+1}\) such that \[ x_1+x_2+\cdots+x_n+x_{n+1} = t. \] \(x_{n+1}\) is the integer to force equality.

Dots and Dividors argument

\(t\) dots and \(n\) dividers. Then the number of solutions is \(\binom{t+n}{n}\). Therefore, the lattice point enumerator of \(\Delta_n\) is \[ L_{\Delta_n}(t) = \binom{t+n}{n} = \frac{(t+1)(t+2)\cdots(t+n)}{n!}. \] Note. This is a polynomial of degree \(n\).

Generating Functions

\(\textbf{Definition. }\)The generating function of a sequence \(a_0, a_1, a_2, \cdots\) is the formal power series \[ f(x) = \sum_{i=0}^\infty a_ix^i. \] Note. Formal power series is a power series which convergence doesn't matter.

\(\textbf{Example. Fibonacci numbers}\) \((f_n)^\infty_{n=0}\) \[ f_0 = 0, f_1 = 1, f_{n+2} = f_{n+1} + f_n. \] The generating function of \(f_n\) is \[ F(x) = \sum_{i=0}^\infty f_ix^i = x + x^2 + 2x^3 + 3x^4 + 5x^5 + \cdots. \] \[ F(x) = x + \sum_{i=2}^\infty f_ix^i, \] \[ F(x) = x + \sum_{i=2}^\infty (f_{i-1}+f_{i-2})x^i, \] \[ F(x) = x + \sum_{i=2}^\infty f_{i-1}x^i + \sum_{i=2}^\infty f_{i-2}x^i, \] \[ F(x) = x + x\sum_{i=1}^\infty f_{i}x^{i-1} + x^2\sum_{i=0}^\infty f_{i}x^{i-2}, \] \[ F(x) = x + xF(x) + x^2F(x), \] \[ F(x) = \frac{x}{1-x-x^2}. \] Note. \(1 - x - x^2 = (1-\varphi x)(1-\phi x)\) where \[ \varphi = \frac{1+\sqrt{5}}{2}, \] \[ \phi = \frac{1-\sqrt{5}}{2}. \]


Ehrhart Series

\(\textbf{Definition. }\)The Ehrhart series of a convex polytope \(P\) is the generating function \[ Ehr_P(x) = 1+ \sum_{i=1}^\infty L_P(i)x^i. \]

\(\textbf{Example. }\)The Ehrhart series of \(\Delta_2\) is \[ Ehr_{\Delta_2}(x) = 1 + \sum_{i=1}^\infty L_{\Delta_2}(i)x^i, \] \[ Ehr_{\Delta_2}(x) = 1 + \sum_{i=1}^\infty \binom{i+2}{2}x^i = \sum_{i=0}^\infty \binom{i+2}{2}x^i, \] \[ Ehr_{\Delta_2}(x) = \sum_{i=0}^\infty (\frac{i^2}{2}+\frac{3}{2}i+1)x^i \] \[ Ehr_{\Delta_2}(x) = \frac{1}{2}\sum_{i=0}^\infty i^2x^i + \frac{3}{2}\sum_{i=0}^\infty i x^i + \sum_{i=0}^\infty x^i \] \[ Ehr_{\Delta_2}(x) = \frac{1}{(1-x)^3} \]

\(\textbf{Example. }\)The Ehrhart series of a lattice polygon \(P\) is \[ Ehr_P(x) = \frac{(A-\frac{B}{2}+1)x^2+(A+\frac{B}{2}-2)x+1}{(1-x)^3}. \]

\(\textbf{Exercise. }\)For \(\Delta_n\subset \mathbb{R}^n\), \[ Ehr_{\Delta_n}(x) = \frac{1}{(1-x)^{n+1}}. \]

Field

Any set that satisfies all following axioms is called a field.

\(\textbf{Definition. }\)[Commutativity of Addition] For all real numbers \(a\) and \(b\), \[ a+b=b+a. \]

\(\textbf{Definition. }\)[Associativity of Addition] For all real numbers \(a\), \(b\), and \(c\), \[ (a+b)+c=a+(b+c). \]

\(\textbf{Definition. }\)[The Additive Identity] There is a real number, denoted by \(0\), such that \[ 0+a=a+0=a, \] for all real numbers \(a\).

\(\textbf{Definition. }\)[The Additive Inverse]: For each real number \(a\), there is a real number \(b\) such that \[ a +b =0. \]

\(\textbf{Definition. }\)[Commutativity of Multiplication] For all real numbers \(a\) and \(b\), \[ ab=ba. \]

\(\textbf{Definition. }\)[The Nontriviality Assumption] \[ 1\neq 0 \]

\(\textbf{Exercise. }\) For \(a\neq 0\) and \(b\neq 0\), show that \((ab)^{-1}=a^{-1}b^{-1}\).

\(\textbf{Proof. }\) Assume that \(a, b\neq 0\) and are in \(\mathbb{R}\), \[ (ab)(ab)^{-1} = 1, \] \[ a a^{-1}bb^{-1} = 1. \] For commutativity of multiplication, \[ a b a^{-1}b^{-1} = 1. \] Thus, \[ ab(ab)^{-1} = a b a^{-1}b^{-1}. \] Therefore, \[ (ab)^{-1} = a^{-1}b^{-1}. \]

\(\textbf{Definition. }\)[Bounded above, Upper bound] Suppose \(S\) is an ordered set, and \(E\subset S\). If there exists a \(\beta\subset S\) such that \(x\leq \beta\) for every \(x\in E\), we say that \(E\) is bounded above, and call \(\beta\) an upper bound of \(E\).

\(\textbf{Definition. }\)[Order] Let \(S\) be a set. An order on \(S\) is a relation, denoted by \(\lt\), with the following two properties: \item (i) If \(x\in S\) and \(y\in S\) then one and only one of the statements \[ x\lt y,\; x=y,\; y\lt x \] is true. \item (ii) If \(x,y,z\in S\),if \(x\lt y\) and \(y\lt z\), then \(x\lt z\).

\(\textbf{Definition. }\)[Order set] An ordered set is a set \(S\) in which an order is defined.

Bounded

\(\textbf{Definition. }\)[Least upper bound/Supremum] Suppose \(S\) is an ordered set, \(E\subset S\), and \(E\) is bounded above. Suppose there exists an \(\alpha\in S\) with the following properties: \item (i) \(\alpha\) is an upper bound of \(E\). \item (ii) If \(\gamma\lt \alpha\) then \(\gamma\) is not an upper bound of \(E\). \\ Then \(\alpha\) is called the least upper bound of \(E\) (that there is at most one such \(\alpha\) is clear from (ii)] or the supremum of \(E\), and we write \[ \alpha= \sup E. \]

\(\textbf{Definition. }\)[Greatest lower bound/Infimum] Suppose \(S\) is an ordered set, \(E\subset S\), and \(E\) is bounded below. Suppose there exists an \(\alpha\in S\) with the following properties: \item (i) \(\alpha\) is an lower bound of \(E\). \item (ii) If \(\alpha\lt \gamma\) then \(\gamma\) is not an lower bound of \(E\). \\ Then \(\alpha\) is called the greatest lower bound of \(E\) or the infimum of \(E\), and we write \[ \alpha= \inf E. \]

\(\textbf{Definition. }\)[Least-upper-bound property] An ordered set \(S\) is said to have the least-upper-bound property if the following is true: \item If \(E\subset S\), \(E\) is not empty, and \(E\) is bounded above, then \(\sup E\) exists in \(S\).

\(\textbf{Theorem. }\) Suppose \(S\) is an ordered set with the least-upper-bound property, \(B\subset S\), \(B\) is not empty, and \(B\) is bounded below. Let \(L\) be the set of all lower bounds of \(B\). Then \[ \alpha = \sup L \] exists in \(S\) and \(\alpha = \inf B\). In particular, \(\inf B\) exists in \(S\).

\(\textbf{Proof. }\) Since \(B\) is bounded below and \(L\) is the set of all lower bounds of \(B\), \(L\neq \emptyset\). Since \(L\) consists of exactly those \(y\in S\) which satisfy the inequality \(y\leq x\) for every \(x\in B\), we see that every \(x\in B\) is an upper bound of \(L\). Thus \(L\) is bounded above. Our hypothesis about \(S\) implies therefore that \(L\) has a supremum in \(S\); call it \(\alpha=\sup L\). If \(\gamma \lt \alpha\) then \(\gamma\) is not an upper bound of \(L\) since \(\alpha\) is the least upper bound. Hence \(\gamma\notin B\). If \(x\lt \alpha\) for some \(x\in B\) and we know that \(x\) is the upper bound of \(L\), then the contradiction exists for \(\alpha\) is the least upper bound of \(L\). Hence, \(\alpha\leq x\) for every \(x\in B\). Thus \(\alpha\in L\) for \(\alpha\) is a lower bound of \(B\). If \(\alpha \lt \beta\) then \(\beta\notin L\), since \(\alpha\) is an upper bound of \(L\). We have shown that \(\alpha\in L\) but \(\beta\notin L\) if \(\beta>\alpha\). In other words, \(\alpha\) is a lower bound of \(B\), but \(\beta\) is not if \(\beta>\alpha\). It shows that \(\alpha=\inf B\).

\(\textbf{Definition. }\)[Ordered Field] An ordered field is a field \(F\) which is also an ordered set, such that \item i \(x+y\lt x+z\) if \(x, y, z\in F\) and \(y\lt z\). \item ii \(xy>0\) if \(x, y\in F\), \(x>0\), \(y>0\).

\(\textbf{Theorem. }\)[Archimedean property of \(\mathbb{R}\)] If \(x\in \mathbb{R}\), \(y\in \mathbb{R}\), and \(x>0\), then there is a positive integer \(n\) such that \[ nx > y. \]

\(\textbf{Proof. }\) I will prove it with mathematical contradiction. Assume there is no \(n\in\mathbb{Z}^+\) such that \[ nx>y. \] In other words, for all \(n\in\mathbb{Z}^+\), \[ nx\leq y. \] Let \(A\) be a set of \(nx\). In that case, \(A\) is bounded above and \(y\) is an upper bound of \(A\). Then \(A\) has an least upper bound and we assume that \(\alpha = \sup A\), and we know that \(nx\leq \alpha\leq y\). Since \(x>0\), \[ \alpha - x \leq \alpha. \] Then we know that \[ \alpha - x \leq mx, \] for some \(m\in\mathbb{Z}^+\). \[ \alpha \leq mx + x, \] \[ \alpha \leq (m + 1)x. \] We know that \(m+1\) is a positive integer. Therefore, contradiction exists.

\(\textbf{Theorem. }\) Every nonempty set of natural numbers has a smallest member.

\(\textbf{Theorem. }\)[\(\mathbb{Q}\) is dense in \(\mathbb{R}\)] If \(x\in \mathbb{R}\), \(y\in \mathbb{R}\),and \(x\lt y\),then there exists a \(p\in\mathbb{Q}\) such that \(x\lt p\lt y\).

\(\textbf{Proof. }\) Let \(a\) and \(b\) be real numbers with \(a\lt b\). First suppose that \(a>0\).By the \ref{Archimedean property}, there is a natural number \(q\) such that \[ q(b-a)>1, \] \[ b-a>\frac{1}{q}. \] Again using the \ref{Archimedean property}, the set of natural numbers \(S\) where \[ S=\{n\in \mathbb{N}\; |\; n/q>b\}, \] is nonempty. According to Theorem 1, \(S\) has a smallest member \(p\). Observe that \(\frac{1}{q}\lt b-a\lt b\lt \frac{n}{q}\) for all \(n\in S\). Since \(\frac{1}{q}\lt \frac{n}{q}\) and \(n\) is a positive integer, \(n>1\), which indicates \(p>1\). Then we can know that \(p-1\) is a natural number (seProblem9) and so, by the minimality of the choice of \(p\), \(\frac{p-1}{q}\lt b\). We also have \[ a=b-(b-a)\lt \frac{p}{q}-\frac{1}{q}=\frac{p-1}{q}. \] Therefore the rational number \(r=\frac{p-1}{q}\) lies between \(a\) and \(b\). \item If \(a\lt 0\), by the \ref{Archimedean property}, there is a natural number \(n\) for which \(n\cdot 1>-a\). Hence, \(n+a>0\). Since \(b>a\), \(n+b>n+a\). We infer from the first case considered that there is a rational number \(r\) such that \(n+a\lt r\lt n+b\). Therefore there exists a rational number \(r-n\) such that \(a\lt r-n\lt b\).

\(\textbf{Theorem. }\) There exists an ordered field \(R\) which has the least-upper-bound property. Moreover, \(R\) contains \(Q\) as a subfield.

\(\textbf{Theorem. }\) For every real \(x > 0\) and every integer \(n > 0\) there is one and only one positive real \(y\) such that \(y^n = x\), this number is written as \(\sqrt[n]{x}\) or \(x^{1/n}\).

\(\textbf{Proof. }\) Let \(E\) be the set consisting of all positive real numbers \(t\) such that \(t^n\lt x\), \(E = \{t>0\;|\;t^n\lt x\}\). If \(t=\frac{x}{1+x}\), then \(0\lt t\lt 1\). Hence \(t^n\lt t\lt x\). Thus \(t\in E\), and we know that \(E\) is not empty. If \(t>1+x\), then \(t^n>t>x\), so that \(t\notin E\). Thus \(1+x\) is an upper bound of \(E\). Hence \ref{Order field R} implies the existence of \[ y = \sup E. \] We know that \[ b^n - a^n=(b-a)(b^{n-1} + b^{n-2}a + \dots + a^{n-1}). \] When \(0\lt a\lt b\), \[ (b-a)(b^{n-1} + b^{n-2}a + \dots + a^{n-1})\lt (b-a)(b^{n-1} + b^{n-1} + \dots + b^{n-1}) = (b-a)\cdot n\cdot b^{n-1}. \] Hence, \[ b^n - a^n \lt (b-a)\cdot n\cdot b^{n-1}. \] Now I will prove that \(y^n = x\) with mathematical contradiction. Assume that \(y^n\lt x\). Choose \(h\) such that \(0\lt h\lt 1\) and \[ h\lt \frac{x-y^n}{n(y+1)^{n-1}}. \] Thus, \(h\cdot n\cdot (y+1)^{n-1}\lt x- y^n\). Put \(a=y\), \(b=y+h\). Then \(0\lt a\lt b\), and \[ (y+h)^n-y^n\lt (y+h-y)\cdot n\cdot (y+h)^{n-1}=h\cdot n\cdot (y+h)^{n-1}\lt h\cdot n\cdot (y+1)^{n-1}\lt x- y^n, \] \[ (y+h)^n-y^n\lt x- y^n, \] \[ (y+h)^n\lt x. \] Thus, \(y+h\in E\). Since \(y+h>y\), and \(y\) is an upper bound of \(E\), contradiction exists. Hence \(y^n\geq x\).\\ Now assume \(y^n > x\). Put \[ k = \frac{y^n-x}{ny^{n-1}}. \] Then, we know that \(k n y^{n-1} = y^n - x\). Since \(n\) is a positive integer and \(x>0\), \[ y^n-x\lt y^n\leq n y^n. \] Thus, \[ 0\lt k = \frac{y^n-x}{ny^{n-1}}\lt \frac{y^n}{ny^{n-1}}\leq \frac{n y^n}{ny^{n-1}} = y, \] \[ 0\lt k\lt y. \] If \(t\geq y-k\), we conclude that \[ y^n- t^n\leq y^n- {(y- k)}^n. \] Since we showed that \[ b^n - a^n \lt (b-a)\cdot n\cdot b^{n-1}, \] when \(0\lt a\lt b\). Thus, \[ y^n- t^n\leq y^n- {(y- k)}^n\lt (y - (y-k))\cdot n\cdot y^{n-1} = k n y^{k-1} = y^n - x, \] \[ y^n- t^n\lt y^n - x, \] \[ - t^n\lt - x, \] \[ t^n>x. \] Hence, \(t\notin E\). For the same reason, \[ y^n- {(y- k)}^n\lt y^n - x, \] \[ - {(y- k)}^n\lt - x, \] \[ {(y- k)}^n > x. \] Hence, \(y-k\) is an upper bound of \(E\). However, \(y=\sup E\) and \(y-k\lt y\) for \(k>0\). Contradiction exists and we can conclude that \(y^n\leq x\). Therefore, \(x\leq y^n\geq y^n\) and \(y^n = x\).


Complex Number

\(\textbf{Definition. }\)[Complex Number] A complex number is an ordered pair \((a, b)\) of real numbers. ''Ordered'' means that \((a, b)\) and \((b, a)\) are regarded as distinct if \(a\neq b\). Let \(x =(a, b)\), \(y =(c, d)\) be two complex numbers. We write \(x=y\) if and only if \(a = c\) and \(b = d\). \[ x+y = (a+c, b+d) \] \[ xy = (ac-bd, ad+bc) \]

\(\textbf{Definition. }\)[\(i\)] \[ i = (0,1) \]

\(\textbf{Theorem. }\) \[ i^2 = -1 \]

\(\textbf{Proof. }\) \[ i^2 = (0,1)(0,1) = (-1, 0) = -1 \]

\(\textbf{Theorem. }\) If \(a\) and \(b\) are real, then \((a, b)=a+bi\).

\(\textbf{Proof. }\) \[ a + bi = (a, 0) + (b, 0)(0, 1) = (a, 0) + (0, b) = (a, b) \]

\(\textbf{Theorem. }\) If \((a, b)(x, y) = 1\) where \((x,y)\neq (0, 0)\), then \((a, b) = (\frac{x}{x^2+y^2}, \frac{-y}{x^2+y^2})\).

\(\textbf{Definition. }\)[Conjugate] If \(a\), \(b\) are real and \(z = a + bi\), then the complex number \(\bar z = a - bi\) is called the conjugate of \(z\). The numbers \(a\) and \(b\) are the real part and the imaginary part of \(z\), respectively. We shall occasionally write \[ a= \Re(z) \] \[ b =\Im(z). \]

\(\textbf{Theorem. }\) If \(w\) and \(z\) \item (a) \(\overline{z+w}=\bar z+\bar w\), \item (b) \(\overline{zw} = \bar{z}\cdot \bar{w}\), \item (c) \(z + \bar z=2\Re(z)\), \(z -\bar z=2i\Im(z)\), \item (d) \(z\bar z\) is real and positive (except when \(z = 0\)).

\(\textbf{Proof. }\) Assume that \(z = a + bi = (a, b)\) and \(w = c + di = (c, d)\). In that case, \(\bar z = a - bi = (a, -b)\) and \(\bar w = c - di = (c, -d)\). \item (a) \[ \overline{z + w} = \overline{(a, b) + (c, d)} = \overline{(a+c, b+d)} = (a+c, -b-d). \] \[ \bar z + \bar w = \overline{(a, b)} + \overline{(c, d)} = (a, -b) + (c, -d) = (a+c, -b-d). \] Hence, \(\overline{z+w}=\bar z+\bar w\). \item (b) \[ \overline{zw} = \overline{(a, b)(c, d)} = \overline{(ac-bd, ad+bc)} = (ac-bd, -ad-bc). \] \[ \bar{z}\cdot\bar{w} = \overline{(a, b)}\overline{(c, d)} = (a, -b)(c, -d) = (ac-bd, -ad-bc). \] Hence, \(\overline{zw} = \bar{z}\cdot \bar{w}\). \item (c) \[ z+\bar z = (a, b) + (a, -b) = (2a, 0) = 2\Re(z). \] \[ z-\bar z = (a, b) + (-a, b) = (0, 2b) = 2i\Im(z). \] \item (d). When \(z\neq 0\), \[ z\cdot\bar z = (a, b)(a, -b) = (a^2+b^2, -ab+ab) = (a^2+b^2, 0). \] Since \(a, b\in\mathbb{R}\) and \(a, b\neq 0\), \(a^2+b^2>0\).

\(\textbf{Definition. }\) If \(z\) is a complex number, its absolute value \(|z|\) is the non-negative square root of \(z\bar z\); that is, \(|z| =(z\bar z)^\frac{1}{2}\).

\(\textbf{Theorem. }\) Let \(z\) and \(w\) be complex numbers. Then \\ (a) \(|z|>0\) unless \(z=0\), \(|0|=0\),\\ (b) \(|\bar z|=|z|\),\\ (c) \(|zw|=|z||w|\),\\ (d) \(|\Re z|\leq |z|\),\\ (e) \(|z+w|\leq |z|+|w|\).\\

\(\textbf{Definition. }\)[Coordinates] For each positive integer \(k\), let \(R^k\) be the set of all ordered \(k\)-tuples \[ x =(x_1, x_2,\dots, x_k), \] where \(x_1,\dots , x_k\) are real numbers, called the coordinates of \(x\). The elements of \(R^k\) are called points, or vectors, especially when \(k > 1\).

\(\textbf{Remark. }\) We shall denote vectors by boldfaced letters. If \(\textbf{y} = (y_1,\dots , y_k)\) and if \(\alpha\) is a real number, put \[ \textbf{x} + \textbf{y} = (x_1 + y_1,\dots , x_k + y_k), \] \[ \alpha\textbf{x}= (\alpha x_1,\dots , \alpha x_k). \] so that \(\textbf{x} + \textbf{y}\in Rk\) and \(\alpha\textbf{x}\in R^k\).

Conclude that every complex number (with one exception!) has two complex square roots.

\(\textbf{Proof. }\) \[ z^2 = (a, b)(a, b) = (a^2-b^2, 2ab). \] \[ a^2 - b^2 = \frac{|w|+u}{2} - \frac{|w|-u}{2}= u. \] \[ 2ab = 2{\left (\frac{(|w|+u)(|w|-u)}{2}\right )}^{1/2} = 2{\left (\frac{|w|^2-u^2}{4}\right )}^{1/2} = 2{\left (\frac{w\bar w-u^2}{4}\right )}^{1/2} = 2{\left (\frac{u^2+v^2-u^2}{4}\right )}^{1/2} = |v|. \] Since \(v\geq 0\), \[ 2ab = |v| = v. \] Hence, \[ z^2 = (u, v) = w, \] for \(v\geq 0\). \[ {\bar z}^2 = (a, -b)(a, -b) = (a^2-b^2, -2ab). \] \[ -2ab = -|v|. \] In this case, \(v\leq 0\). Thus, \[ -2ab = -(-v) = v. \] Hence, \[ {\bar z}^2 = (u, v) = w, \] for \(v\leq 0\). When \(w = 0\), there only exists one square root, which is \(0\).

Finite, Countable, and Uncountable Sets

\(\textbf{Definition. }\) Let \(A\) and \(B\) be two sets and let \(f\) be a mapping of \(A\) into \(B\). If \(E\subset A\), \(f(E)\) is defined to be the set of all elements \(f(x)\), for \(x\in E\). We call \(f(E)\) the image of \(E\) under \(f\). In this notation, \(f(A)\) is the range of \(f\). It is clear that \(f(A)\subset B\). If \(f(A) = B\), we say that \(f\) maps \(A\) onto \(B\). (Note that, according to this usage, onto is more specific than into.)

\(\textbf{Definition. }\)[Equivalence Relation] If there exists a \(1-1\) mapping of \(A\) onto \(B\), we say that \(A\) and \(B\) can be put in \(1-1\) correspondence, or that \(A\) and \(B\) have the same cardinal number, or, briefly, that A and B are equivalent, and we write \(A\sim B\). This relation clearly has the following properties:\\ It is reflexive: \(A\sim A\).\\ It is symmetric: If \(A\sim B\), then \(B\sim A\).\\ It is transitive: If \(A\sim B\) and \(B\sim C\), then \(A\sim C\).

\(\textbf{Definition. }\) For any positive integer \(n\), let \(J_n\) be the set whose elements are the integers \(1, 2,\dots , n\); let \(J\) be the set consisting of all positive integers. For any set \(A\), we say:\\ (a) \(A\) is \textbf{finite} if \(A\sim J_n\) for some \(n\) (the empty set is also considered to be finite).\\ (b) \(A\) is \textbf{infinite} if \(A\) is not finite.\\ (c) \(A\) is \textbf{countable} if \(A\sim J\).\\ (d) \(A\) is \textbf{uncountable} if \(A\) is neither finite nor countable. \\ (e) \(A\) is \textbf{at most countable} if \(A\) is finite or countable.\\

\(\textbf{Note. }\) Countable sets are sometimes called enumerable, or denumerable.

\(\textbf{Definition. }\)[Sequence] By a sequence, we mean a function \(f\) defined on the set \(J\) of all positive integers. If \(f(n)=x_n\), for \(n\in J\),it is customary to denote the sequence \(f\) by the symbol \(\{x_n\}\), or sometimes by \(x_1\), \(x_2\) , \(x_3,\dots\) The values of \(f\), that is, the elements \(x_n\), are called the terms of the sequence. If \(A\) is a set and if \(x_n\in A\) for all \(n\in J\), then \(\{x_n\}\) is said to be a sequence in \(A\), or a sequence of elements of \(A\).

\(\textbf{Note. }\) The terms \(x_1, x_2, x_3, \dots\) of a sequence need not be distinct.

\(\textbf{Theorem. }\) Every infinite subset of a countable set \(A\) is countable.

\(\textbf{Proof. }\) Suppose \(E\subset A\), and \(E\) is infinite. Arrange the elements \(x\) of \(A\) in a sequence \(\{x_n\}\) of distinct elements. Construct a sequence \(\{n_k\}\) as follows: Let \(n_1\) be the smallest positive integer such that \(x_{n_1}\in E\). Having chosen \(n_1, \dots , n_{k-1}\) \((k =2, 3, 4, ...)\), let \(n_k\) be the smallest integer greater than \(n_{k-1}\) such that \(x_{n_k}\in E\). Putting \(f(k) = x_{n_k}\) \((k = 1, 2, 3, ...)\), we obtain a 1-1 correspondence between \(E\) and \(J\).

\(\textbf{Note. }\) Countable sets represent the ''smallest'' infinity: No uncountable set can be a subset of a countable set.

\(\textbf{Definition. }\) Let \(A\) and \(\Omega\) be sets, and suppose that with each element \(\alpha\) of \(A\) there is associated a subset of \(n\) which we denote by \(E_\alpha\).\\ The set whose elements are the sets \(E_\alpha\) will be denoted by \(\{E_\alpha\}\). Instead of speaking of sets of sets, we shall sometimes speak of a collection of sets, or a family of sets.\\ The union of the sets \(E_\alpha\) is defined to be the set \(S\) such that \(x\in S\) if and only if \(x\in E_\alpha\) for at least one \(\alpha\in A\). We use the notation \begin{equation} S = \bigcup_{\alpha\in A} E_\alpha. \end{equation} If \(A\) consists of the integers \(1, 2, \dots , n\), one usually writes \begin{equation} S = \bigcup_{m=1}^{n} E_m \end{equation} or \begin{equation} S = E_1\cup E_2\cup\dots\cup E_n. \end{equation} If \(A\) is the set of all positive integers, the usual notation is \begin{equation} S = \bigcup_{m=1}^{\infty} E_m. \end{equation} The intersection of the sets \(E_\alpha\) is defined to be the set \(P\) such that \(x\in P\) if and only if \(x\in E_\alpha\) for every \(\alpha\in A\). We use the notation \begin{equation} P = \bigcap_{\alpha\in A} E_\alpha, \end{equation} or \begin{equation} S =\bigcap_{m=1}^{n} E_m=E_1\cap E_2\cap\dots\cap E_n, \end{equation} or \begin{equation} S =\bigcap_{m=1}^{\infty} E_m, \end{equation} as for unions. If \(A\cap B\) is not empty, we say that \(A\) and \(B\) intersect; otherwise they are disjoint.

\begin{eg} Let \(A\) be the set of real numbers \(x\) such that \(0\lt x\leq 1\). For every \(x\in A\), let \(E_x\) be the set of real numbers \(y\) such that \(0\lt y\lt x\). Then \item (a) \(E_x\subset E_z\) if and only if \(0\lt x\leq z\leq 1\).

\(\textbf{Proof. }\) If \(E_x\subset E_z\), then for every \(a\in E_x\) is in \(E_z\). Since \(E_x = \{u\in\mathbb{R}\;|\;0\lt u\lt x\}\) and \(E_z = \{u\in\mathbb{R}\;|\;0\lt u\lt z\}\), we can know that \(x\leq z\). Thus, \(0\lt x\leq z\leq 1\). \item (b) \(\bigcup_{x\in A} E_x = E_1\).

Metric Space

\(\textbf{Definition. }\)[Metric spaces] A metric space \((X, d)\) is a space \(X\) of objects (called points), together with a distance function or metric \(d: X\times X \rightarrow [0, +\infty)\), which associates to each pair \(x,y\) of points in \(X\) a non-negative real number \(d(x, y) \geq 0\). Furthermore, the metric must satisfy the following four axioms: \item (a) For any \(x\in X\), we have \(d(x, x) = 0\). \item (b) (Positivity) For any distinct \(x, y\in X\), we have \(d(x, y) > 0\). \item (c) (Symmetry) For any \(x, y\in X\), we have \(d(x, y) = d(y, x)\). \item (d) (Triangle inequality) For any \(x, y, z\in X\), we have $d(x, z) \leq d(x, y) + d(y, z)$.

\(\textbf{Definition. }\)[Metric Space] A set \(X\), whose elements we shall call points, is said to be a metric space if with any two points \(p\) and \(q\) of \(X\) there is associated a real number \(d(p, q)\), called the distance from \(p\) to \(q\), such that \item (a) \(d(p,q)>0\) if \(p\neq q\); \(d(p,p)=0\); \item (b) \(d(p, q) = d(q, p)\); \item (c) \(d(p,q)\leq d(p,r)+d(r,q)\), for any \(r\in X\). Any function with these three properties is called a \textbf{distance function}, or a \textbf{metric}.

\(\textbf{Remark. }\)Every subset of a euclidean space is a metric space.


\(\textbf{Definition. }\)[Segment] By the \textbf{segment} \((a, b)\) we mean the set of all real numbers \(x\) such that \(a\lt x\lt b\).

\(\textbf{Definition. }\)[Interval] By the interval \([a, b]\) we mean the set of all real numbers \(x\) such that \(a\leq x\leq b\). Occasionally we shall also encounter "half-open intervals" \([a, b)\) and \((a, b]\); the first consists of all \(x\) such that \(a\leq x\lt b\), the second of all \(x\) such that \(a\lt x\leq b\).

\(\textbf{Definition. }\)[\(k\)-cell] If \(a_i\lt b_i\) for \(i=1,\dots,k,\) the set of all points \(x=(x_1,\dots, x_k)\) in \(R^k\) whose coordinates satisfy the inequalities \(a_i\leq x_i\leq b_i\) \((1\leq i\leq k)\) is called a \(k\)-cell. Thus a \(1\)-cell is an interval, a \(2\)-cell is a rectangle, etc.

\(\textbf{Definition. }\)[Open Ball (or Closed Ball)] If \(x\in R^k\) and \(r > 0\), the open (or closed) ball \(B\) with center at \(x\) and radius \(r\) is defined to be the set of all \(y\in R^k\) such that \(|y-x|\lt r\) (or \(|y-x|\leq r\)).

\(\textbf{Definition. }\) We call a set \(E\subset R^k\) convex if \[ \lambda x+(1-\lambda)y\subset E \] whenever \(x\in E\), \(y\in E\), and \(0\lt \lambda\lt 1\).

\(\textbf{Theorem. }\)Balls are convex.

\(\textbf{Proof. }\)For if \(|y-x|\lt r\), \(|z-x|\lt r\), and \(0\lt \lambda\lt 1\), we have \[ \begin{align*} |\lambda(y-x)+(1-\lambda)(z-x)| & =\lambda|y- x|+(1-\lambda)|z- x|\\ &\lt \lambda r+(1-\lambda)r\\ &=r\\ \end{align*} \]

\(\textbf{Remark. }\)The same proof applies to closed balls. It is also easy to see that \(k\)-cells are convex.


\(\textbf{Definition. }\)[Neighborhood] In a metric \(X\), a neighborhood of \(p\) is a set \(N_r(p)\) consisting of all \(q\) such that \(d(p, q)\lt r\), for some \(r > 0\), The number \(r\) is called the radius of \(N_r(p)\).

\(\textbf{Definition. }\)[Limit Point] In a metric \(X\), A point \(p\) is a limit point of the set \(E\) if every neighborhood of \(p\) contains a point \(q\neq p\) such that \(q\in E\).

\(\textbf{Definition. }\)[Isolated Point] In a metric \(X\), if \(p\in E\) and \(p\) is not a limit point of \(E\), then \(p\) is called an isolated point of \(E\).

\(\textbf{Definition. }\)[Closed set] In a metric \(X\), \(E\) is closed if every limit point of \(E\) is a point of \(E\).

\(\textbf{Definition. }\)[Interior point] A point \(p\) is an interior point of \(E\) if there is a neighborhood \(N\) of \(p\) such that \(N\subset E\).

\(\textbf{Definition. }\)[Open set] In a metric \(X\), \(E\) is open if every point of \(E\) is an interior point of \(E\).

\(\textbf{Definition. }\)[Complement] The complement of \(E\) (denoted by \(E^c\)) is the set of all points \(p\in X\) such that \(p\notin E\).

\(\textbf{Definition. }\)[Perfect set] In a metric \(X\), \(E\) is perfect if \(E\) is closed and if every point of \(E\) is a limit point of \(E\).

\(\textbf{Definition. }\)[Bounded] In a metric \(X\), \(E\) is bounded if there is a real number \(M\) and a point \(q\in X\) such that \(d(p, q) \lt M\) for all \(p\in E\).

\(\textbf{Definition. }\)[Dense] In a metric \(X\), \(E\) is dense in \(X\) if every point of \(X\) is a limit point of \(E\), or a point of \(E\)(or both).


\(\textbf{Remark. }\) In \(R^1\), neighborhoods are segments, whereas in \(R^2\) neighborhoods are interiors of circles.


\(\textbf{Theorem. }\)Every neighborhood is an open set.

\(\textbf{Proof. }\)Consider a neighborhood \(E =N_r(p)\), and \(q\in E\). Then there is a \(h>0\) and \(h\in R\) such that \(d(p,q)=r-h\). For all points \(s\) such that \(d(q, s) \lt h\), they form a neighborhood \(N_h(q)\). \[ d(p,s)\leq d(p,q)+d(q,s)\lt r-h+h=r, \] so that all \(s\) are in \(E\), which implies that \(N_h(q)\subset N_r(p)=E\). Thus \(q\) is an interior point of \(E\). And every point in \(E\) is an interior point.


\(\textbf{Theorem. }\) If \(p\) is a limit point of a set \(E\), then every neighborhood of \(p\) contains infinitely many points of \(E\).

\(\textbf{Proof. }\) I will prove it with mathematical contradiction. Assume that \(p\) is a limit point of \(E\) and there is a neighborhood \(N\) of \(p\) which contains only a finite number of points of \(E\). Let \(q_1,\dots , q_n\) be those points of \(N\cap E\), which are distinct from \(p\), and put \[ r = \min_{1\leq m\leq n} d(p, q_m). \] [We use this notation to denote the smallest of the numbers \(d(p, q_1),\dots , d(p, q_n)\).] The minimum of a finite set of positive numbers is clearly positive, so that \(r > 0\). The neighborhood \(N_r(p)\) contains no point \(q\) of \(E\) such that \(q\neq p\). Since \(x\) is a limit point of the set \(S\) if every neighborhood of \(x\) contains a point \(y\neq x\) such that \(y\in S\). It shows that \(p\) is not a limit point of \(E\). Contradiction exists.

\(\textbf{Theorem. }\) A finite point set has no limit points.

\(\textbf{Proof. }\) I will prove it with mathematical contradiction. If there exists a limit point \(p\) in a finite set \(E\), then the neighborhood \(N\) of \(p\) contains finitely many points, which are in \(E\). However, \(E\) is a finite set, so contradiction exists.

\(\textbf{Theorem. }\) Let \(\{E_\alpha\}\) be a (finite or infinite) collection of sets \(E_\alpha\). Then \[ \left(\bigcup_\alpha E_\alpha\right)^c = \bigcap_\alpha (E_a^c). \]

\(\textbf{Proof. }\) If \(x\in \left(\bigcup_\alpha E_\alpha\right)^c\), then \(x\notin \bigcup_\alpha E_\alpha\); hence \(x\notin E_\alpha\) for any \(\alpha\), hence \(x\in E_\alpha^c\) for every \(\alpha\), so that \(x\in\bigcap_\alpha (E_a^c)\). Thus, \[ \left(\bigcup_\alpha E_\alpha\right)^c\subset \bigcap_\alpha (E_a^c). \] Conversely, if \(x\in\bigcap_\alpha (E_a^c)\), then \(x\in E_a^c\) for every \(\alpha\), hence \(x\notin E_\alpha\) for any \(\alpha\), hence \(x\notin\bigcup_\alpha E_\alpha\), so that \(x\in \Bigg(\bigcup_\alpha E_\alpha\Bigg)^c\). Thus, \[ \bigcap_\alpha (E_a^c)\subset \left(\bigcup_\alpha E_\alpha\right)^c. \] It follows that \[ \left(\bigcup_\alpha E_\alpha\right)^c = \bigcap_\alpha (E_a^c). \]


\(\textbf{Theorem. }\) Let \(\{E_\alpha\}\) be a (finite or infinite) collection of sets \(E_\alpha\). Then \[ \left(\bigcap_\alpha E_\alpha\right)^c = \bigcup_\alpha (E_a^c). \]

\(\textbf{Proof. }\) If \(x\in \left(\bigcap_\alpha E_\alpha\right)^c\), then \(x\notin \bigcap_\alpha E_\alpha\); hence \(x\notin E_\alpha\) for some \(\alpha\), hence \(x\in E_\alpha^c\) for some \(\alpha\), so that \(x\in\bigcup_\alpha (E_a^c)\). Thus, \[ \left(\bigcap_\alpha E_\alpha\right)^c\subset \bigcup_\alpha (E_a^c). \] Conversely, if \(x\in\bigcup_\alpha (E_a^c)\), then \(x\in E_a^c\) for some \(\alpha\), hence \(x\notin E_\alpha\) for some \(\alpha\), hence \(x\notin\bigcap_\alpha E_\alpha\), so that \(x\in \Bigg(\bigcap_\alpha E_\alpha\Bigg)^c\). Thus, \[ \bigcup_\alpha (E_a^c)\subset \left(\bigcap_\alpha E_\alpha\right)^c. \] It follows that \[ \left(\bigcap_\alpha E_\alpha\right)^c = \bigcup_\alpha (E_a^c). \]


\(\textbf{Theorem. }\) A set \(E\) is open if and only if its complement is closed.

\(\textbf{Proof. }\) First, suppose \(E^c\) is closed. Choose \(x\in E\). Then \(x\notin E^c\). Since \(E^c\) is closed, which means it contains all its limit points, thus \(x\) is not a limit point of \(E^c\). Hence there exists a neighborhood \(N\) of \(x\) such that \(N\cap E^c\) is empty, that is, \(N\subset E\). Thus \(x\) is an interior point of \(E\), and every \(x\in E\) is an interior point of \(E\), which shows that \(E\) is open.\\ Next, suppose \(E\) is open. Let \(x\) be a limit point of \(E^c\). Then every neighborhood of \(x\) contains a point of \(E^c\), which means there is no neighborhood \(N\) of \(x\) such that \(N\subset E\), so that \(x\) is not an interior point of \(E\). Since \(E\) is open, this means that \(x\in E^c\). It follows that \(E^c\) is closed for \(E^c\) contains all of its limit points.


\(\textbf{Theorem. }\) For any collection \(\{G_\alpha\}\) of open sets, \(\bigcup_\alpha G_\alpha\) is open.

\(\textbf{Proof. }\) Put \(G=\bigcup_\alpha G_\alpha\). If \(x\in G\), then \(x\in G_\alpha\) for some \(\alpha\). Since \(G_\alpha\) is open, \(x\) is an interior point of \(G_\alpha\), and \(x\) is also an interior point of \(G\). Thus, \(G\) is open.


\(\textbf{Theorem. }\) For any collection \(\{F_\alpha\}\) of closed sets, \(\bigcap_\alpha F_\alpha\) is closed.

\(\textbf{Proof. }\) Since we know that \[ \Bigg(\bigcap_\alpha F_\alpha\Bigg)^c = \bigcup_\alpha (F_a^c), \] and \(F_\alpha^c\) is open for \(F_\alpha\) is closed. Based on the previous theorem, we know that \(\bigcup_\alpha (F_a^c)\) is open, which implies that \(\Bigg(\bigcap_\alpha F_\alpha\Bigg)^c\) is open. Therefore, \(\bigcap_\alpha F_\alpha\) is closed.


\(\textbf{Theorem. }\) For any finite collection \(G_1,\dots ,G_n\) of open sets, \(\bigcap_{i=1}^n G_i\) is open.

\(\textbf{Proof. }\) Assume that \(G = \bigcap_{i=1}^n G_i\). For any \(x\in G\), \(x\in G_i\) for all \(i\in\{1, \dots, n\}\), and there exists an Neighborhood \(N_i\) of \(x\) with radius \(r_i\) such that \(N_i\subset G_i\) where \(i=1, \dots, n\) for \(G_i\) is open. Choose \[ r = \min(r_1, r_2, \dots, r_n). \] For any \(x\), let \(N\) be a neighborhood of \(x\) with radius \(r\). Then \(N\subset G_i\) for all \(i\). Therefore, \(N\subset G\) for \(G=\bigcap_{i=1}^n G_i\). It shows that any point in \(G\) is an interior point, and it means that \(G\) is open.


\(\textbf{Theorem. }\) For any finite collection \(F_1, \dots, F_n\) of closed sets, \(\bigcup_{i=1}^n F_n\) is closed.

\(\textbf{Proof. }\) Since we know that \[ \Bigg(\bigcup_\alpha F_\alpha\Bigg)^c = \bigcap_\alpha (F_a^c), \] and \(F_1^c, F_2^c, \dots, F_n^c\) are all open. Based on the previous theorem, \(\bigcap_{i=1}^n F_i\) is open. Thus, \(\Bigg(\bigcup_{i=1}^n F_i\Bigg)^c\) is open. Therefore, \(\bigcup_{i=1}^n F_i\) is closed.


\(\textbf{Definition. }\)[Closure] If \(X\) is a metric space, if \(E\subset X\), and if \(E'\) denotes the set of all limit points of \(E\) in \(X\), then the closure of \(E\) is the set \(\bar{E} = E\cup E'\).


\(\textbf{Theorem. }\) If \(X\) is a metric space and \(E\subset X\), then \(\bar{E}\) is closed.

\(\textbf{Proof. }\) Since \(\bar{E} = E\cup E'\) where \(E'\) is the set of limit points of \(E\), if \(x\in P\) and \(x\notin \bar{E}\), then \(x\) is not a limit point of \(E\) and \(x\) is not in \(E\). Thus, \(x\in \bar{E}^c\) and there exists a neighborhood \(N\) of \(x\) such that \(N\cap E=\emptyset\). Thus, for any \(x\in \bar{E}^c\), there exists an neighborhood \(N\) such that \(N\subset \bar{E}^c\), which implies that \(\bar{E}^c\) is open. Hence, \(\bar{E}\) is closed.


\(\textbf{Theorem. }\) If \(X\) is a metric space and \(E\subset X\), then \(\bar{E} =E\) if and only if \(E\) is closed.

\(\textbf{Proof. }\) Based on previous theorem, we know that if \(E = \bar{E}\) then \(E\) is closed since \(\bar{E}\) is closed. Conversely, if \(E\) is closed, then \(E\) contains all its limit points (based on the definition of closed sets). Hence, \(E'\subset E\). Since \(\bar{E} = E\cup E'\), \(\bar{E} = E\).


\(\textbf{Theorem. }\) If \(X\) is a metric space and \(E\subset X\), then \(\bar{E}\subset F\) for every closed set \(F\subset X\) such that \(E\subset F\).

\(\textbf{Proof. }\) Since \(F\) is closed, \(F'\subset F\), which implies that every limit point of \(F\) is a point of \(F\) (based on the definition). Since \(E\subset F\), every limit point of \(E\) is a point of \(F\), which means that \(E'\subset F\). Therefore \(E'\cup E\subset F\), which is \(\bar{E}\subset F\).


\(\textbf{Theorem. }\) If \(X\) is a metric space and \(E\subset X\), then \(\bar{E}\) is the smallest closed subset of \(X\) that contains \(E\).

\(\textbf{Proof. }\) Since \(\bar{E}=E\cup E'\), \(E\subset \bar{E}\) and \(\bar{E}\) is closed. For any closed set \(F\) containing \(E\), we know that \(\bar{E}\subset F\). Therefore, \(\bar{E}\) is the smallest closed subset of \(X\) that contains \(E\).


\(\textbf{Theorem. }\) Let \(E\) be a nonempty set of real numbers which is bounded above. Let \(y=\sup E\). Then \(y\in \bar{E}\). Hence \(y\in E\) if \(E\) is closed.

\(\textbf{Proof. }\) If \(y\in E\), then \(y\in\bar{E}\) for \(\bar{E}=E\cup E'\). Assume \(y\notin E\). For every \(h>0\) \(y-h\) will not be an upper bound of \(E\), which means there exists then a point \(x\in E\) such that \(y-h\lt x\lt y\). It shows that any neighborhood of \(y\) contains some points which are not \(y\) in \(E\). Thus \(y\) is a limit point of \(E\). Hence \(y\in \bar{E}\).


\(\textbf{Definition. }\)[Open relative] Suppose \(E\subset Y\subset X\), where \(X\) is a metric space. \(E\) is open relative to \(Y\) if to each \(p\in E\) there is associated an \(r>0\) such that \(q\in E\) whenever \(d(p, q) \lt r\) and \(q\in Y\).

\(\textbf{Remark. }\) \((a, b)\) where \(a\lt b\in R\) is open in \(R\) but it is not open in \(R^2\).

\(\textbf{Theorem. }\) Suppose \(Y\subset X\). A subset \(E\) of \(Y\) is open relative to \(Y\) if and only if \(E = Y\cap G\) for some open subset \(G\) of \(X\).

\(\textbf{Proof. }\) Firstly, suppose \(E\) is open relative to \(Y\). To each \(p\in E\) there is a positive number \(r_p\) such that the conditions \(d(p, q)\lt r_p\), \(q\in Y\) imply that \(q\in E\). Let \(V_p\) be the set of all \(q\in X\) such that \(d(p, q) \lt r_p\), and define \[ G = \bigcup_{p\in E} V_p. \] Then \(G\) is an open subset of \(X\), since every neighborhood is an open set and, for any collection \(\{G_\alpha\}\) of open sets, \(\bigcup_\alpha G_\alpha\) is open. Since \(p\in V_p\) for all \(p\in E\), \(E\subset G\cap Y\). Since \(V_p\subset E\), we have \(V_p\cap Y\subset E\) for every \(p\in E\). Since \(G =\bigcup_{p\in E} V_p\), \(G\cap Y\subset E\). Thus \(E = G\cap Y\).\\ Secondly, suppose that \(G\) is open in \(X\) and \(E = G\cap Y\), every \(p\in E\) has a neighborhood \(V_p\subset G\). Then \(V_p\cap Y\subset E\), so that \(E\) is open relative to \(Y\).


Compact set

\(\textbf{Definition. }\)[Open cover] By an open cover of a set \(E\) in a metric space \(X\) we mean a collection \({G_\alpha}\) of open subsets of \(X\) such that \(E\subset \bigcup_\alpha G_a\).

\(\textbf{Definition. }\)[Compact set] A subset \(K\) of a metric space \(X\) is said to be compact if every open cover of \(K\) contains a finite subcover. i.e, if \(\{G_\alpha \}\) is an open cover of \(K\), then there are finitely many indices \(\alpha_1, \dots, \alpha_n\) such that \(K\subset G_{a_1}\cup\dots\cup G_{a_n}\).

\(\textbf{Remark. }\) Every finite set is compact.

\(\textbf{Definition. }\)[Compact relative] A subset \(K\) of a metric space \(X\) is said to be compact relative to \(K\) if every open cover of \(K\) contains a finite subcover. i.e, if \(\{G_\alpha \}\) is an open cover of \(K\), then there are finitely many indices \(\alpha_1, \dots, \alpha_n\) such that \(K\subset G_{a_1}\cup\dots\cup G_{a_n}\).


\(\textbf{Theorem. }\) Suppose \(K\subset Y\subset X\). Then \(K\) is compact relative to \(X\) if and only if \(K\) is compact relative to \(Y\).

\(\textbf{Proof. }\) Suppose \(K\) is compact relative to \(X\), and let \(\{V_\alpha\}\) be a collection of sets, open relative to \(Y\), such that \(K\subset U_\alpha V_\alpha\). Since a subset \(E\) of \(Y\) is open relative to \(Y\) if and only if \(E = Y\cap G\) for some open subset \(G\) of \(X\), for any \(\alpha\), there are sets \(G_\alpha\), open relative to \(X\), such that \(V_\alpha = Y\cap G_\alpha\); and since \(K\) is compact relative to \(X\), open cover of \(K\) contains a finite subcovers. Hence, we have \[ K\subset G_{\alpha_1}\cup\cdots\cup G_{\alpha_n}, \] for some finitely many indices \(\alpha_1, \dots, \alpha_n\). Since \(K\subset Y\), \[ K\subset (Y\cap G_{\alpha_1})\cup\dots\cup(Y\cap G_{\alpha_n}), \] \[ K\subset V_{\alpha_1}\cup\cdots\cup V_{\alpha_n}, \] This proves that \(K\) is compact relative to \(Y\).\\ Conversely, suppose \(K\) is compact relative to \(Y\), let \(\{G_\alpha\}\) be a collection of open subsets of \(X\) which covers \(K\), and put \(V_\alpha = Y\cap G_\alpha\). Thus, \(V_\alpha\) is open relative to \(Y\) and \[ K\subset V_{\alpha_1}\cup\cdots\cup V_{\alpha_n}, \] for some choice of \(\alpha_1, \dots, \alpha_n\). Since \(V_a\subset G_\alpha\), \[ K\subset G_{\alpha_1}\cup\cdots\cup G_{\alpha_n}. \]

\(\textbf{Theorem. }\) Compact subsets of metric spaces are closed.

\(\textbf{Proof. }\) Let \(K\) be a compact subset of a metric space \(X\). We shall prove that \(K^c\) is an open subset of \(X\).\\ Suppose \(p\in X\), \(p\notin K\). If \(q\in K\), let \(V_q\) and \(W_q\) be neighborhoods of \(p\) and \(q\), respectively, of radius less than \(\frac{1}{2}d(p, q)\). Thus, \(V_q\cap W_q=\emptyset\). Since \(K\) is compact, there are finitely many points \(q_1, \cdots, q_n\) in \(K\) such that \[ K\subset W_{q_1} \cup\dots\cup W_{q_n} = W. \] If \(V=V_{q_1}\cap\dots\cap V_{q_n}\), then \(V\) is a neighborhood of \(p\) which does not intersect \(W\). Thus, for any \(p\in X\) and \(p\notin X\), there is a open neighborhood of \(p\) in \(K^c\). Thus, \(K^c\) is open and \(K\) is closed. theorem follows.


\(\textbf{Theorem. }\) Closed subsets of compact sets are compact.

\(\textbf{Proof. }\) Suppose that \(F\subset K\subset X\). \(K\) is compact and \(F\) is closed relative to \(X\). Let \(\{V_\alpha\}\) be an open cover of \(F\), thus \(F\subset \bigcup_\alpha V_\alpha\). Since \(F\) is closed in \(X\), \(F^c\) is open in \(X\). Since \(F^c\cup F = X\), \(K\subset (F\cup F^c)\subset (\bigcup_\alpha V_\alpha)\cup F^c\). Thus, \(\{V_\alpha, F^c\}\) is an open cover of \(K\). Since \(K\) is compact, \(F\subset K\subset (\bigcup_{i=1}^n V_{\alpha_i} \cup F^c)\). Since \(F\cap F^c=\emptyset\), \(F\subset (\bigcup_{i=1}^n V_{\alpha_i} \cup F^c)\) indicates that \(F\subset \bigcup_{i=1}^n V_{\alpha_i}\) for any \(\{V_\alpha\}\). Therefore, \(F\) is compact.


\(\textbf{Corollary. }\)If \(F\) is closed and \(K\) is compact, then \(F\cap K\) is compact.

\(\textbf{Proof. }\) Since \(K\) is compact, \(K\) is closed. Since for any collection \(\{F_\alpha\}\) of closed sets, \(\bigcap_\alpha F_\alpha\) is closed. Hence, \(F\cap K\) is closed. Since \(F\cap K\subset K\), and closed subsets of compact sets are compact, \(F\cap K\) is compact.

We write \(p_n\), or \[ \lim_{n\rightarrow\infty} p_n = p. \] If \(\{p_n\}\) does not converge, it is said to diverge.

\(\textbf{Theorem. }\) Let \(\{p_n\}\) be a sequence in a metric space \(X\). \(\{P_n\}\) converges to \(p\in X\) if and only if every neighborhood of \(p\) contains \(p_n\) for all but finitely many \(n\).

\(\textbf{Proof. }\) Suppose \(p_n\rightarrow p\) and let \(V\) be a neighborhood of \(p\). For some \(\epsilon>0\), the conditions \(d(q, p) \lt \epsilon\), \(q\in X\) imply \(q\in V\). Corresponding to this \(\epsilon\), there exists \(N\) such that \(n\geq N\) implies \(d(p_n, p) \lt \epsilon\). Thus \(n\geq N\) implies \(p_n\in V\). Conversely, suppose every neighborhood of p contains all but finitely many of \(n\) the \(P_n\). Fix \(e>0\), and let \(V\) be the set of all \(q\in X\) such that \(d(p, q) \lt e\). By assumption, there exists \(N\) (corresponding to this \(V\)) such that \(P_n\in V\) if \(n\geq N\). Thus \(d(p_n,p)\lt e\) if \(n\geq N\); hence \( P_n >p\).

Cauchy Sequence

\(\textbf{Definition. }\)[Cauchy Sequence] A sequence \(\{p_n\}\) in a metric space \(X\) is said to be a Cauchy sequence if for every \(\epsilon > 0\) there is an integer \(N\) such that \(d(p_n , p_m) \lt \epsilon\) if \(n\geq N\) and \(m\geq N\).

\(\textbf{Definition. }\)[Diameter] Let \(E\) be a nonempty subset of a metric space \(X\), and let \(S\) be the set of all real numbers of the form \(d(p, q)\), with \(p\in E\) and \(q\in E\). The sup (least upper bound) of \(S\) is called the diameter of \(E\).

\(\textbf{Remark. }\) If \(\{p_n\}\) is a sequence in \(X\) and if \(E_N\) consists of the points \(P_N, P_{N+1} ,P_{N+2}, \dots\), it is clear from the two preceding definitions that \({p_n}\) is a Cauchy sequence if and only if \[ \lim diam E_N= 0. \]


\(\textbf{Theorem. }\) If \(\overline{E}\) is the closure of a set \(E\) in a metric space \(X\), then \[ diam\overline{E} = diam E. \]

\(\textbf{Proof. }\) Since \(E\subset \overline{E}\), \[ diam E\leq diam\overline{E}. \] Fix \(\epsilon > 0\), and choose \(p\in \overline{E}\), \(q\in \overline{E}\). By the definition of \(E\), there are limit points points \(p'\), \(q'\), in \(E\) such that \(d(p, p') \lt \epsilon\), \(d(q, q') \lt \epsilon\). At the same time, \(d(p',q')\leq diam\overline{E}\). Since \(E\) is in a metric space \(X\), \[ d(p, q)\leq d(p, q')+ d(q', q)\leq d(p, p') + d(p', q') + d(q', q) \lt 2\epsilon + d(p', q')\leq 2\epsilon + diam E. \] It follows that \(diam \overline{E}\leq 2\epsilon + diam E\), and since \(\epsilon\) was arbitrary, \(diam\overline{E}\leq diam E\). Therefore, \(diam E=diam\overline{E}\).

\(\textbf{Definition. }\)[\(k\)-Manifold] Let \(k > 0\). Suppose that \(M\) is a subspace of \(\mathbb{R}^n\) having the following property: For each \(p\in M\), there is a set \(V\) containing \(p\) that is open in \(M\), a set \(U\) that is open in \(\mathbb{R}^k\) , and a continuous map \(a : U\rightarrow V\) carrying \(U\) onto \(V\) in a one-to-one fashion, such that: \item (1) \(a\) is of class \(C^r\). \item (2) \(a^{-1}: V\rightarrow U\) is continuous. \item (3) \(Da(x)\) has rank \(k\) for each \(x\in U\). Then \(M\) is called a \(k\)-manifold without boundary in \(\mathbb{R}^n\), of class \(C^r\). The map \(a\) is called a coordinate patch on \(M\) about \(p\).
